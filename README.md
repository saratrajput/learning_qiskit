# Learning Qiskit


## Installation
* Create a minimal environment with Anaconda
```
$ conda create -n name_of_my_env python=3.7
$ source activate name_of_my_env
```

* Install qiskit with pip
```
pip install qiskit
```

* Additional packages:
```
$ pip install matplotlib
$ pip install jupyter
$ pip install ipywidgets
$ pip install seaborn
$ pip install pygments
```
